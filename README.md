# Social Connect 🌐

A full-featured social media app built with React Native & Expo.

## Features

### Week 1-3 (Core)
- ✅ User Authentication (Signup/Login)
- ✅ Bottom Tab Navigation
- ✅ Home Feed with Posts
- ✅ Like/Unlike with Animation
- ✅ Comments
- ✅ Notifications (in-app)
- ✅ Profile Edit
- ✅ Settings (Dark Mode, Change Password)

### Week 4-6 (Advanced)
- ✅ Follow/Unfollow System
- ✅ Direct Messaging/Chat
- ✅ Post Edit & Delete
- ✅ Image Upload in Posts
- ✅ User Search
- ✅ Profile Picture Upload
- ✅ Performance Optimizations (FlatList)

## Tech Stack
- React Native
- Expo
- Expo Router
